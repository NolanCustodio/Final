A short summary on the change from IPv4 to IPv6.

Internet protocol version 4, or more commonly known as IPv4, is the current most popular used protocol version. There are many reasons as to why there is a push to start using IPv6, but to name an important one, there are not enough IP addresses in the IPv4 system. The reason why there are not enough IP addresses in the IPv4 system is that the format that is used when making an IP address is xxxxxxxx.xxxxxxxx.xxxxxxxx.xxxxxxxx where when making an actual address one would replace every x with either a 1 or 0. This is where IPv6 comes into play and improves the system. Using IPv4 there are 2^32 possible IP addresses which was thought to be enough but, recently it has been found that there are too many computers and that they are not enough IP addresses. IPv6 uses the format xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx:xxxx where replacing every x with the hexadecimal number system, 0 - 9 and a,b,c,d,e,f. By using this there are many times more addresses that can be used. This solves a very daunting problem that the technological world was faced with, but this was only one advancement that IPv6 has to offer.
